Написать CRUD приложение

В приложении должна быть страница на которую выводятся все юзеры с
Возможностью добавлять, удалять и изменять юзера.
У пользователя может быть роль user или admin
Сделать логин через Google
Добавить фильтр,  который после логина будет отправлять пользователей с ролью user на страницу /user,
а пользователей с ролью admin на страницу /admin на которой будет таблица всех юзеров с возможностью их изменения
Сделать так чтобы user не мог заходить на все страницы которые начинаются на /admin
(т.e. Не мог видеть таблицу пользователей или производить с ней какие-либо действия)

Используемые технологии:
Spring Boot
Spring MVC
Spring Data JPA
Spring Security
Oauth2 (Google)
Hibernate
Bootstrap
Thymeleaf
